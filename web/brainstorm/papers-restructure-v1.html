Search-Augmented Language Models

Reader / PDF view · page 4 · Method section

3. Retrieval and Evidence Use

The model retrieves passages before generation and uses them as a working memory for answer construction. The important assumption is that the retrieved evidence is both relevant and sufficiently grounded for the final response.

A failure appears when retrieval confidence is high but the evidence only supports part of the claim. The answer must expose which source supports which sentence.

A. Papers 중심으로 ARES 재정렬

제품명은 ARES로 두고, 첫 경험과 탭 구조를 논문 리딩 중심으로 바꿉니다.

  • 기존 Lab 자산을 살림
  • 배포 리스크가 가장 낮음
  • 추천 방향

B. Papers를 독립 제품처럼 전면화

ARES의 실험/작성 축을 뒤로 미루고 Papers 브랜드와 구조를 우선합니다.

  • 정체성이 가장 선명함
  • 기존 문서와 라벨 변경 범위 큼

C. 기존 ARES에 Wiki/Agent만 추가

현 Read/Lab/Insight/Write를 유지하고 Wiki와 Agent를 하위 패널로 넣습니다.

  • 가장 작은 변경
  • 논문 읽기 중심성이 약해질 수 있음
